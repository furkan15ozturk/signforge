# SignForge

A modular Sign Language Translation (SLT) research framework for experimenting across multiple architectures, attention mechanisms, and training strategies — all controlled via a single config file.

## Features
- Config-driven training: swap encoder, loss, and dataset without touching code
- Dataset support: PHOENIX-2014T, CSL-Daily
- Encoder variants: Vanilla Transformer, Deformable Attention, GlossAttention
- Auxiliary losses: SBERT Knowledge Transfer (more coming)
